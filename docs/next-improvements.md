# Next Improvements

План после MVP-релиза. Порядок можно менять по результатам закрытого теста.

## Gameplay

- Добавить восстановление HP за токены или по времени.
- Добавить расширенные эффекты навыков.
- Добавить PvE-режим с сериями боев.
- Добавить рейтинговые сезоны и leaderboard.
- Добавить защиту от фарма одного и того же бота сверх текущих мультипликаторов.

## Arena

- Добавить предпросмотр вероятного диапазона силы соперников.
- Добавить симулятор боя для администратора.
- Добавить больше типов событий в лог боя.
- Добавить визуальную replay-страницу боя.
- Добавить настройку правил матчмейкинга по лигам.

## Economy

- Расширить магазин категориями и ротацией товаров.
- Добавить продажу или разбор предметов.
- Добавить крафт только после стабилизации экономики.
- Добавить источники предметов после боя.
- Добавить отдельные цены для сервисов в настройках баланса.

## Inventory And Items

- Добавить durability и ремонт.
- Добавить сравнение экипировки перед надеванием.
- Добавить массовые действия в инвентаре.
- Добавить журнал движения предметов для спорных ситуаций.

## Admin

- Добавить dashboard по активности игроков и боям.
- Добавить просмотр конкретных боев из админки.
- Добавить ручную выдачу предметов игроку.
- Добавить импорт/экспорт справочников.
- Добавить предпросмотр итоговых SPECIAL для вида и предмета.

## Operations

- Добавить GitHub Actions или GitLab CI.
- Добавить автоматический запуск тестов и сборки на pull request.
- Добавить deploy script.
- Добавить backup-инструкцию для PostgreSQL.
- Добавить error monitoring.
- Добавить health-check endpoint.

## UX

- Довести темную тему до устойчивого Fallout/Morrowind-настроения.
- Довести светлую тему до Arcanum/Sacred-настроения.
- Добавить компактные мобильные панели для арены, магазина и инвентаря.
